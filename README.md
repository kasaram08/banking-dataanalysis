# 🏦 Banking Customer Analysis Project

## 📌 Project Overview
Analyzed **11,162 real bank customers** to identify deposit patterns 
and provide actionable marketing recommendations.

## 🎯 Business Problem
Only 47.38% of customers made deposits. Identified which customer 
segments are most likely to subscribe to term deposits.

## 🔍 Key Findings
- Total Customers Analyzed: **11,162**
- Average Age: **41.2 years**
- Average Balance: **₹1,528**
- Deposit Rate: **47.38%**
- Best Segment: **Students (74.72% deposit rate)**
- 688 customers have **negative balance**

## 💡 Recommendations
1. Target students — highest deposit rate (74.72%)
2. Launch special student savings account
3. Create retirement savings scheme
4. Debt recovery program for 688 customers
